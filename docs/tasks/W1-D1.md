# W1-D1 — Bootstrap Godot Project

> 必读：`docs/tasks/CONTEXT.md`

## 目标
让 Godot 工程能开 + Autoload 注册 + Input Map 配好 + 一个能 F5 跑的 Hello World 场景。

## 必读文件
- `project.godot`（当前内容）
- `src/scripts/core/game_manager.gd`（要被 autoload 的目标）
- `README.md` 的"开发环境"段落

## 步骤

### 1. [CODE] 修改 `project.godot`，加 autoload 段

在 `[application]` 段里加：
```
run/main_scene="res://src/scenes/main.tscn"
```

文件末尾加新段：
```
[autoload]

GameManager="*res://src/scripts/core/game_manager.gd"
```

> **注意**：路径前的 `*` 表示作为 Singleton autoload（区别于普通节点 autoload）。

### 3. [CODE] 创建 `src/scenes/main.tscn`

写入：
```
[gd_scene format=3 uid="uid://bxc3y8z6m4nqp"]

[node name="Main" type="Node2D"]

[node name="Hello" type="Label" parent="."]
offset_left = 300.0
offset_top = 800.0
offset_right = 800.0
offset_bottom = 900.0
text = "Hello Metro Escape"
```

> uid 可任意，只要每个 .tscn 唯一即可（也可省略，Godot 会补）。

## 验收标准

- [ ] `project.godot` 含 `[autoload]` 段，注册 `GameManager`
- [ ] `project.godot` 含 `[input]` 段，含 7 个 action：move_left/right/up/down, fire, reload, interact
- [ ] `project.godot` 的 `[application]` 段含 `run/main_scene="res://src/scenes/main.tscn"`
- [ ] `src/scenes/main.tscn` 存在且是合法 Godot 4 .tscn 格式
- [ ] 没改 `src/scripts/` 下任何 GDScript 文件

## 常见坑

1. `[autoload]` 路径必须以 `*` 开头（singleton 模式），否则 GameManager 无法被全局访问
2. `.tscn` 第一行 `[gd_scene format=3]` 必须有 — Godot 4 不接受 format 2
3. interact action 的 keycode 占位用了 S，**不是错误**，是为了保证不冲突；编辑器里可改
4. project.godot 是 INI，每段间需要空行
