## 步骤

### 1. [CODE] 创建 `src/scripts/world/bullet.gd`

```gdscript
class_name Bullet
extends Area2D

var direction: Vector2 = Vector2.RIGHT
var speed: float = 1500.0
var damage: int = 30
var max_distance: float = 600.0

var _traveled: float = 0.0


func _ready() -> void:
	body_entered.connect(_on_body_entered)
	area_entered.connect(_on_area_entered)


func _physics_process(delta: float) -> void:
	var step := direction * speed * delta
	global_position += step
	_traveled += step.length()
	if _traveled >= max_distance:
		queue_free()


func _on_body_entered(body: Node) -> void:
	if body.has_method("take_damage"):
		body.take_damage(damage)
	queue_free()


func _on_area_entered(area: Area2D) -> void:
	# Hit other bullets / pickups should be ignored
	if area is Bullet:
		return
```

### 3. [CODE] 创建 `src/scripts/world/dummy.gd`

```gdscript
class_name Dummy
extends StaticBody2D

@export var max_hp: int = 100

var hp: int = 100


func _ready() -> void:
	hp = max_hp
	add_to_group("enemy")  # so auto-aim in D5 can find it


func take_damage(amount: int) -> void:
	hp = max(0, hp - amount)
	modulate = Color(1, 0.3, 0.3, 1)  # flash red on hit
	if hp == 0:
		queue_free()
```

## 常见坑

1. Bullet 是 Area2D **不是** RigidBody2D — 不要让物理引擎管它
2. `body_entered` 信号收 PhysicsBody2D（包括 StaticBody2D 的 dummy 和 player 的 CharacterBody2D）；如果只想打 enemy，用 collision layer/mask 过滤（W2 任务）
3. `area_entered` 是其他 Area2D（如其他子弹、撤离区）— 别误伤
4. Bullet 无 max_distance 检测会永远存在 — 必须有 _traveled 累计
5. Weapon.try_fire 期望 bullet_scene 已配置 — ak.tres 里必须设置
